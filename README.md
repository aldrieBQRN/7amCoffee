# 7AM Coffee - Point of Sale (POS) System

A web-based Point of Sale and Inventory Management System designed for 7AM Coffee. This application streamlines cafe operations by handling order processing, inventory tracking, and sales reporting in a responsive interface.

## Features

### User Roles & Authentication
* **Secure Login:** Role-based access control (Owner vs. Staff).
* **Owner/Admin Dashboard:** Full access to analytics, inventory, and sales reports.
* **Staff Point of Sale:** Restricted access focused on order processing.

### Point of Sale (POS)
* **Interactive Menu:** Visual product grid with category filtering (Coffee, Pasta, Cake, etc.).
* **Real-time Cart:** Add items, adjust quantities, and calculate totals instantly.
* **Stock Validation:** Prevents adding items that are out of stock.
* **Checkout System:** Supports multiple payment methods (Cash, GCash, PayMaya).
* **Digital Receipt:** Generates a printable receipt upon successful checkout.

### Inventory Management
* **CRUD Operations:** Add, update, and delete menu items.
* **Stock Monitoring:** Visual indicators for "Low Stock" and "Out of Stock" items.
* **Categories:** Organize items by type (e.g., Salad, Snacks, Frappuccino).

### Analytics & Reporting
* **Sales Dashboard:** View today's sales, total transactions, and average order value.
* **Transaction History:** Detailed log of all past orders with date, items, and payment method.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla), Bootstrap 5
* **Backend:** PHP
* **Database:** MySQL
* **Icons:** Bootstrap Icons
